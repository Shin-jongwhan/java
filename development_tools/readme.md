# 개발도구
### 개발도구는 왜 써야 할까?
#### 땅은 손, 삽, 굴삭기 모두가 팔 수 있고 본질적인 기능은 같다.
#### 도구라는 것은 기능을 좀 더 쉽고 효율적으로 할 수 있게 만들어주는 것이다.
#### 개발 도구도 똑같다.
### <br/><br/><br>

## 이클립스 (IDE)
#### IDE : 개발에 필요한 여러가지 도구들이 하나로 결합된 것. 파이썬 IDE 도 그런 기능 !
#### 소스 에디터, 컴파일러, 디버거, 실행까지 가능한 도구이다.
