## API 와 API 문서 보는 법
### System.out.println("a") 와 같은 것은 어떻게 구성되어 있는지
- System 은 클래스이다.
- println() 은 메쏘드이다.
- 그럼 중간에 out 이 남는데,, 얘는 System 클래스 안에 있는 객체 (object) 이다.
- 그러니까 println() 은 out 에 포함된 메쏘드이다.
