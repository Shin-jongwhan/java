### 다음과 같이 변수 타입이 달라도 연산은 가능하다.
```
public class variable_double {
    public static void main(String[] args) {
        double a;
        a = 1.2 + 1;        // 가능
        System.out.println(a + 1);      // 가능
    }
}
```
